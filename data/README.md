# AI-Powered Retail Analytics Dashboard

## Overview

This project demonstrates how AI can transform traditional business intelligence workflows.

### Traditional Workflow

Business User → Analyst → BI Developer → Dashboard → Insight

Time Required: Days or Weeks

### AI-Powered Workflow

Business User → AI → Dashboard → Insight

Time Required: Minutes

The objective of this project is to showcase how modern AI tools can generate executive-ready dashboards directly from live cloud-hosted datasets, reducing analytics turnaround time and enabling faster decision-making.

---

## Project Components

### Retail Sales Dataset

Contains:

* Monthly sales performance
* Product-level metrics
* Revenue
* Units sold
* Pricing information
* Promotion indicators

### Price Elasticity Dataset

Contains:

* SKU-level elasticity coefficients
* Sensitivity classifications
* Confidence ratings
